==========
trame-code
==========

Widget for Monaco VS code editor for trame

Development
-----------------------------------------------------------

Build and install the Vue components

.. code-block:: console

    cd vue-gantt
    npm i
    npm run build-bundle
    cd -

Install the python library

.. code-block:: console
    python3 -m venv .venv
    . .venv/bin/activate
    pip install -e .

Launch example

.. code-block:: console
   python3 example/gantt-test.py
